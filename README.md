<h2> :triangular_flag_on_post: How to run </h2>

```bash
# Clonar o repositório
git clone https://github.com/guilhermecapitao/proffy-discovery

# Entrar no diretório
cd proffy-discovery

# Baixar as dependências
npm i

# Executar o servidor
npm run dev
```
